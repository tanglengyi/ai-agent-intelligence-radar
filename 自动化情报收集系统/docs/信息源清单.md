## 当前 RSSHub 状态

- GitHub Release 源已配置 RSSHub 路由，并增加 GitHub 官方 `releases.atom` 兜底；公开 RSSHub 返回 403 时仍可抓取。
- 公众号源已配置 RSSHub 路由，但公开 `rsshub.app` 可能返回 403 或限流。长期使用建议自建 RSSHub，然后在 `.env` 里设置：

```bash
RSSHUB_BASE_URL=https://你的-rsshub-域名
```

- 国内平台公告目前多数使用网页监控兜底，后续可以逐步替换成更稳定的 RSSHub 路由或页面 diff。
